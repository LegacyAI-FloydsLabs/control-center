1. Tame Chrome — Biggest Win (~2-3 GB recoverable)

Chrome is running 28 tab renderers plus GPU, network, and extension processes — 35 total, eating ~4.2 GB. Close tabs you're not actively using, or install a tab suspender extension like The Great Suspender. Consider moving low-priority browsing to Safari, which uses significantly less memory per tab on macOS. Alternatively, use Chrome profiles to separate work vs. personal and only open one at a time.
